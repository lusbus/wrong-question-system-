# 公考错题助手

公务员考试错题解析与学习提升系统

## 项目简介

这是一款专为公务员考试考生设计的桌面应用程序，通过 AI 智能分析错题，提供结构化的五步复盘，帮助考生快速提升薄弱环节。

## 核心功能

### 📝 智能题目解析

- 粘贴题目内容（无需包含答案），AI 自动解析并给出正确答案
- 智能识别题干、选项和正确答案
- 支持行测和申论两大科目

### 🎯 五步复盘分析

1. **拆结构**：分析文段层次结构，明确指出重心句
2. **分析错因**：7 种错因类型精确定位错误原因
3. **对比选项**：标准公考排除术语逐项分析
4. **避坑口诀**：一句话总结教训，快速记忆
5. **同类题标记**：标记题型陷阱，提供自查动作

### 📊 学习统计分析

- 错误类型分布可视化
- 薄弱知识点追踪
- AI 个性化学习建议
- 复习进度统计

### 📖 结构化笔记导出

- 自动生成 Markdown 格式的错题笔记
- 支持 Obsidian 等笔记软件导入
- 知识图谱关联

### 🤖 多 AI 提供商支持

- DeepSeek（推荐）
- 阿里云（通义千问）
- 智谱 AI
- 百度文心
- 其他兼容 OpenAI API 的服务

## 快速开始

### 环境要求

- Node.js >= 16
- npm >= 8

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

### 构建安装包

```bash
npm run build
```

构建完成后，安装包位于 `dist-release/` 目录。

## 使用指南

### 1. 配置 AI 服务

- 打开设置页面
- 选择 AI 提供商
- 填写 API Key 和模型名称
- 保存配置

### 2. 录入错题

- 在"录入"页面选择科目和题型
- 粘贴题目内容（包含题干和选项，不要包含正确答案）
- 点击"AI 智能解析题目"

### 3. 选择答案

- 查看 AI 解析的题目信息
- 在下方选择你自己的答案
- 点击"开始 AI 五步复盘"

### 4. 查看分析结果

- 查看五步复盘分析结果
- 确认无误后点击"复盘正确，保存"

### 5. 查看统计

- 在"统计"页面查看学习进度
- 获取 AI 生成的个性化学习建议

## 项目结构

```
src/
├── main/                    # Electron 主进程
│   ├── database/           # 数据库相关
│   ├── services/           # 服务层（AI、Markdown 等）
│   ├── ipc/                # IPC 通信处理
│   └── utils/              # 工具函数
├── renderer/               # React 渲染进程
│   ├── components/         # 组件
│   ├── pages/              # 页面
│   ├── services/           # API 服务
│   ├── stores/             # 状态管理
│   └── utils/              # 工具函数
└── shared/                 # 共享代码
    └── types.ts            # 类型定义

database/                   # 数据库初始化脚本
docs/                       # 项目文档
dist-release/               # 构建输出
```

## 数据模型

### 错题 (WrongQuestion)

- 题目内容、科目、题型
- 用户答案、正确答案
- 错误类型、错因分析
- 五步复盘各字段
- 知识点、相似题目

### AI 配置 (AIConfig)

- 提供商、API Key
- 模型名称、Base URL
- 激活状态
