# 公务员考试错题解析与学习提升系统 - 设计文档

**日期**: 2026-05-14  
**版本**: Phase 1  
**状态**: 待审核

---

## 1. 项目概述

为备考公务员考试的学生开发一套 Windows 桌面应用，具备错题解析、笔记生成、知识点提炼、解题思路总结、举一反三训练和学习建议生成六大核心功能。系统采用 Markdown 文件存储错题本，支持实时更新的思维导图和错题索引。

---

## 2. 技术栈

| 层级 | 技术选型 | 说明 |
|------|----------|------|
| 框架 | Electron 28+ | 跨平台桌面应用框架 |
| 前端 | React 18 + TypeScript | UI 组件库 |
| 状态管理 | Zustand | 轻量级状态管理 |
| UI 组件 | Ant Design | 企业级 UI 组件库 |
| 数据库 | SQLite (better-sqlite3) | 本地数据存储 |
| AI 服务 | OpenAI API / 通义千问 | 错题解析与内容生成 |
| 思维导图 | Mermaid.js | Markdown 内嵌思维导图渲染 |
| 构建工具 | Vite + Electron Builder | 开发与打包 |

## 5. 数据库设计

### 5.1 wrong_questions（错题表）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | 自增主键 |
| subject | TEXT | 科目（行测/申论） |
| question_type | TEXT | 题型分类 |
| question_content | TEXT | 题目内容 |
| user_answer | TEXT | 用户答案 |
| correct_answer | TEXT | 正确答案 |
| error_type | TEXT | 错误类型 |
| error_analysis | TEXT | AI 错误分析 |
| correct_solution | TEXT | 正确解法 |
| knowledge_points | TEXT JSON | 知识点列表 |
| similar_questions | TEXT JSON | 举一反三题目 |
| status | TEXT | 状态（待复习/已掌握） |
| source | TEXT | 题目来源 |
| created_at | DATETIME | 创建时间 |
| updated_at | DATETIME | 更新时间 |

### 5.2 knowledge_points（知识点表）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | 自增主键 |
| name | TEXT | 知识点名称 |
| subject | TEXT | 所属科目 |
| category | TEXT | 知识点分类 |
| parent_id | INTEGER FK | 父知识点 ID |
| description | TEXT | 知识点描述 |
| created_at | DATETIME | 创建时间 |

### 5.3 study_notes（学习记录表）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | 自增主键 |
| question_id | INTEGER FK | 关联错题 ID |
| note_content | TEXT | 笔记内容 |
| review_count | INTEGER | 复习次数 |
| last_reviewed | DATETIME | 最后复习时间 |
| created_at | DATETIME | 创建时间 |

### 5.4 ai_config（AI 配置表）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | 自增主键 |
| provider | TEXT | AI 提供商 |
| api_key | TEXT | API 密钥 |
| model | TEXT | 模型名称 |
| base_url | TEXT | API 基础 URL |
| is_active | BOOLEAN | 是否启用 |

## 6. AI 提示词设计

### 6.1 错误分析 Prompt

```
你是一名公务员考试辅导专家。请分析以下错题，按照以下格式输出：

【错误类型】（从以下选择：知识点掌握不足/解题思路偏差/审题失误/计算错误/时间管理不当/心理因素）
【错误分析】详细分析错误原因（100-200字）
【正确解法】标准解题步骤
【核心知识点】列出3-5个相关知识点
【解题技巧】该题型的通用解题思路

题目：{question}
用户答案：{user_answer}
正确答案：{correct_answer}
题型：{question_type}
```

### 6.2 举一反三 Prompt

```
你是一名公务员考试出题专家。请根据以下错题，生成3道相似题目：

要求：
1. 考察相同或相关知识点
2. 难度相近
3. 题目类型一致
4. 包含答案和解析

原错题：{question}
涉及知识点：{knowledge_points}
```

### 6.3 学习建议 Prompt

```
你是一名公务员考试学习规划师。请根据学生的错题情况，生成个性化学习建议：

错题统计：
- 总错题数：{total}
- 各题型错误分布：{distribution}
- 薄弱知识点：{weak_points}
- 错误类型分布：{error_types}

请输出：
1. 本周学习重点（3条）
2. 推荐复习顺序
3. 针对性训练建议
4. 时间分配建议
```

## 7. 功能模块设计

### 7.1 错题录入模块

- 支持文本直接输入
- 支持图片粘贴（自动 OCR 识别）
- 智能题型分类
- 科目选择（行测/申论）
- 来源标注（国考/省考/模拟）

### 7.2 AI 解析模块

- 自动调用 AI 分析错误原因
- 生成结构化解析结果
- 提取核心知识点
- 支持手动修正 AI 结果

### 7.3 错题笔记模块

- 自动生成 Markdown 格式笔记
- 实时更新思维导图
- 错题索引自动维护
- 支持手动编辑补充

### 7.4 知识图谱模块

- 知识点自动提取与分类
- 知识点关联网络构建
- Mermaid 思维导图生成
- 图谱可视化展示

### 7.5 举一反三模块

- AI 生成相似题目
- 难度分级选择
- 一键加入练习队列
- 答题结果记录

### 7.6 学习建议模块

- 薄弱知识点分析
- 个性化学习计划
- 复习时间提醒
- 进度跟踪统计

## 8. 实时更新机制

### 8.1 触发时机

- 新增错题时
- 修改错题状态时
- 删除错题时
- 复习完成时

### 8.2 更新流程

```
新增错题
  → 更新 SQLite 数据库
  → 更新对应类型错题本.md
  → 更新类型思维导图.md
  → 更新总览/错题总览.md
  → 更新知识点关联
```

### 8.3 并发控制

- 文件写入使用队列机制
- 避免同时写入同一文件
- 提供手动同步按钮

## 10. 错误处理

- AI API 调用失败：重试机制 + 降级提示
- 数据库操作失败：事务回滚 + 用户提示
- 文件写入失败：自动备份 + 手动恢复
- 图片 OCR 失败：手动输入回退
