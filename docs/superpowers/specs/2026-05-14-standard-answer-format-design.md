# 公考标准答案格式改进设计

## 目标
让 AI 分析结果更符合公考标准答案的格式，层次分明、逻辑清晰、专业术语准确。

## 改进内容

### 1. 文段分析部分
- 按文段顺序引用原文（"首先指出...接着介绍...尾句总结强调..."）
- 明确指出文段结构类型（总分、分总、总分总等）
- 点明重点/主旨句

### 2. 选项分析部分
- 逐项分析每个选项
- 使用公考标准排除术语：
  - "非重点，排除"
  - "范围缩小/扩大，排除"
  - "无中生有，排除"
  - "偏离文意，排除"
  - "强加因果，排除"
  - "偷换概念，排除"

### 3. 正确答案
- 简洁总结：故正确答案为 X

### 4. 错误类型和知识点
- 保持原有格式，但要求更简洁专业

## 实现位置
- 修改文件：`src/main/utils/prompts.ts`
- 修改函数：`PROMPTS.analyzeError`
