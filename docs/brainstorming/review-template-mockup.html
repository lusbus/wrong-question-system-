📚 公考错题助手 - 错题复盘模板优化方案

当前系统

1. 录入错题

用户粘贴题目内容 + 输入自己的答案

2. AI 分析

AI 生成错误类型、错误分析、正确答案、核心知识点

缺少拆结构分析 缺少错因分类

3. 纠错机制

用户可输入正确答案，AI 重新分析

4. 保存结果

保存到数据库 + 生成 Markdown 错题本

问题：分析结果偏简单，没有结构化的复盘流程，用户难以深度理解错题

🟢 优化后系统

1. 录入错题

用户粘贴题目内容 + 输入自己的答案 + 输入正确答案

新增：正确答案输入

2. AI 结构化分析

AI 生成五步复盘结果：

  • 拆结构（层次/作用/重心句）
  • 分析错因（8种错因类型分类）
  • 对比选项（逐项分析对错原因）
  • 提炼避坑口诀
  • 同类题标记
新增：结构化复盘

3. 用户交互

用户可确认、纠错、或要求 AI 重新分析

4. 保存 + 生成错题复盘卡

保存到数据库 + 生成结构化 Markdown 复盘卡

改进：Markdown 模板升级
✅ 优势：结构化复盘流程，帮助用户深度理解错题，提炼避坑口诀

📊 拆结构分析

  • 将原文拆分成多个层次
  • 标注每层的作用（铺垫/过渡/重心）
  • 提供判断重心句的三种方法
  • 帮助用户理解文段逻辑结构

🎯 错因分类系统

  • 8种标准错因类型供选择
  • 把背景当主旨 / 以偏概全 / 无中生有
  • 过度推断 / 偷换概念 / 只看表面
  • 转折没抓住 / 其他自定义错因

📝 选项对比表

  • 逐项分析每个选项
  • 说明为什么对 / 为什么错
  • 使用标准排除术语
  • 帮助用户建立系统排除法

💡 避坑口诀

  • 用一句话总结这道题的教训
  • 以后做题能快速想起来
  • 积累个人专属避坑指南
  • 定期复习加深记忆

🔗 同类题标记

  • 标记这道题和之前哪道题一样
  • 记录自查动作
  • 建立错题关联网络
  • 避免重复犯同样的错误

📈 错题统计分析

  • 按错因类型统计
  • 按题型分布统计
  • 识别高频错误模式
  • 生成个性化学习建议
